# bookstore
Book Store POC

# RUN Client 

1. Make a .env file in client folder
2. Populate REACT_APP_API_URL in env
3. Example: REACT_APP_API_URL=http://localhost:5000/api/v1
4. npm install 
5. npm start

# RUN Server
1. Make a .env file in server folder
2. Populate .env file in server folder
3. Example: 
    TOKEN_SECRET=random
    TOKEN_SECRET_EXP=1d
4. npm install
5. npm start
