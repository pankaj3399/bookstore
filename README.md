# bookstore
Book Store POC

# RUN Client 

Make a .env file in client folder
Populate REACT_APP_API_URL in env
Example: REACT_APP_API_URL=http://localhost:5000/api/v1
npm install 
npm start

# RUN Server
Make a .env file in server folder
Populate .env file in server folder
Example: 
TOKEN_SECRET=random
TOKEN_SECRET_EXP=1d
npm install
npm start
